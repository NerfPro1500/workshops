Mechanics Demo

Arrow Keys / WASD: Move  |  Space / Up / W: Jump  |  Z: Shoot

Designer's Lens

This demo shows progressive discovery: the player starts with one ability (jump), finds a powerup that grants a second (double jump), then the new ability opens a path to a third (shoot). Notice how each pickup is placed right where you need to use the new ability — the game teaches through placement, not through text. The HUD on the left shows what mechanics are locked or unlocked at any moment.
